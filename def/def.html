HTML

HTML (HyperText Markup Language) est le langage de balisage standard pour créer des pages web et des applications web. Avec HTML, vous pouvez structurer le contenu d'une page web en utilisant une variété de balises, telles que <div>, <p>, <h1>, etc. HTML fournit les blocs de construction de base pour le développement web.

CSS

CSS (Cascading Style Sheets) est un langage de feuille de style utilisé pour décrire la présentation d'un document écrit en HTML. Avec CSS, vous pouvez contrôler la mise en page, les couleurs, les polices et d'autres aspects visuels d'une page web. CSS permet aux développeurs web de créer des conceptions visuellement attrayantes et réactives.

JavaScript

JavaScript est un langage de programmation de haut niveau couramment utilisé avec HTML et CSS pour ajouter de l'interactivité et un comportement dynamique aux pages web. Avec JavaScript, vous pouvez manipuler le contenu d'une page web, répondre aux actions des utilisateurs, valider les entrées des formulaires, et bien plus encore. C'est l'une des technologies de base du développement web.
