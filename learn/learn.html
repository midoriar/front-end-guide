Balises HTML

<h1>

La balise <h1> définit un titre de niveau supérieur.

Exemple :

<h1>Ceci est un titre</h1>
<p>

La balise <p> définit un paragraphe.

Exemple :

<p>Ceci est un paragraphe</p>
<a>

La balise <a> définit un hyperlien.

Exemple :

<a href="https://example.com">Visitez notre site</a>
<img>

La balise <img> définit une image.

Exemple :

<img src="image.jpg" alt="Image">
<div>

La balise <div> définit une division ou une section dans un document HTML.

Exemple :

<div>Ceci est une division</div>
<span>

La balise <span> est utilisée pour regrouper des éléments en ligne dans un document.

Exemple :

<p>Mon nom est <span>Jean</span></p>
<ul>

La balise <ul> définit une liste non ordonnée.

Exemple :

<ul><li>Élément 1</li><li>Élément 2</li></ul>
<ol>

La balise <ol> définit une liste ordonnée.

Exemple :

<ol><li>Premier élément</li><li>Deuxième élément</li></ol>
<li>

La balise <li> définit un élément de liste.

Exemple :

<ul><li>Élément 1</li><li>Élément 2</li></ul>
<table>

La balise <table> définit un tableau.

Exemple :

<table>
<tr><th>En-tête 1</th><th>En-tête 2</th></tr>
<tr><td>Donnée 1</td><td>Donnée 2</td></tr>
</table>

Propriétés CSS

color

La propriété color définit la couleur du texte.

Exemple :

color: blue;
background-color

La propriété background-color définit la couleur de fond d'un élément.

Exemple :

background-color: yellow;
font-size

La propriété font-size définit la taille de la police.

Exemple :

font-size: 16px;
font-family

La propriété font-family définit la famille de polices à utiliser pour le texte.

Exemple :

font-family: Arial, sans-serif;
margin

La propriété margin définit la marge d'un élément.

Exemple :

margin: 10px;
padding

La propriété padding définit le rembourrage d'un élément.

Exemple :

padding: 20px;
border

La propriété border définit la bordure d'un élément.

Exemple :

border: 1px solid black;
width

La propriété width définit la largeur d'un élément.

Exemple :

width: 100px;
height

La propriété height définit la hauteur d'un élément.

Exemple :

height: 100px;
display

La propriété display spécifie le comportement d'affichage d'un élément.

Exemple :

display: flex;
position

La propriété position spécifie le type de positionnement utilisé pour un élément.

Exemple :

position: absolute;
top

La propriété top spécifie la position supérieure d'un élément.

Exemple :

top: 50px;
left

La propriété left spécifie la position gauche d'un élément.

Exemple :

left: 50px;
right

La propriété right spécifie la position droite d'un élément.

Exemple :

right: 50px;
bottom

La propriété bottom spécifie la position inférieure d'un élément.

Exemple :

bottom: 50px;
float

La propriété float spécifie comment un élément doit flotter.

Exemple :

float: left;
clear

La propriété clear spécifie quels éléments peuvent flotter à côté de l'élément dégagé et de quel côté.

Exemple :

clear: both;
